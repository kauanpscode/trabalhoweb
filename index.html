Embora tenha explorado várias vertentes, o desenvolvimento web é a área que mais me atrai. Gosto de criar interfaces intuitivas e funcionais que proporcionem uma ótima experiência ao usuário. Atualmente, estou focado em aprimorar minhas habilidades em JavaScript, Node.js e Python, sempre em busca de novas oportunidades para aprender e crescer como desenvolvedor.

Estou sempre aberto a novos desafios que me permitam expandir meu horizonte no campo da tecnologia.

Portfólio

Primeiro Projeto: Landing Page

Este foi o meu primeiro projeto solo, onde criei uma landing page utilizando apenas HTML e CSS. O objetivo era aplicar os conhecimentos adquiridos no curso de JavaScript, focando na estruturação e estilização de páginas web. Esse projeto foi uma ótima oportunidade para explorar os fundamentos do desenvolvimento web e compreender melhor como as diferentes partes de um site se conectam.

Com este projeto, aprendi muito sobre a importância de uma boa organização do código e a necessidade de pensar na experiência do usuário desde o início. A página criada é simples, mas funcional, e me deu a base necessária para avançar em projetos mais complexos.

Calculadora de IMC

Este projeto consistiu na criação de uma calculadora de Índice de Massa Corporal (IMC), uma aplicação interativa que permite aos usuários inserirem seus dados e obterem instantaneamente o valor de seu IMC. Desenvolvido utilizando JavaScript, o projeto me ajudou a entender melhor como manipular dados de entrada do usuário e como utilizar lógicas de programação para gerar resultados dinâmicos.

A calculadora de IMC não é apenas uma ferramenta prática, mas também um excelente exemplo de como a programação pode ser usada para resolver problemas do cotidiano. Este projeto me deu uma boa compreensão de como as variáveis, funções e eventos funcionam em JavaScript.
